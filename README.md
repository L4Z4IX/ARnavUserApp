# AR Navigation User App

Ez a repó az Android alkalmazás, azaz megjelenítő alklamazás része az ARNav projektnek. A webkiszolgáló része megtalálható a https://github.com/L4Z4IX/ARnavServer repóban.

Az alkalmazás feladata a felhasználói és adminisztrátori megjelenítési felület biztosítása. Tartalmaz adminisztrátorok számára adatbeviteli, GPS mérési és adattesztelő felületet, a felhasználóknak pedig navigációs cél választást és magát a navigációt.

Az alkalmazás futtatható Andorid Studio-n keresztül, vagy a kibocsájtott https://github.com/L4Z4IX/ARnavUserApp/releases/tag/1.0 APK segítségével.

Az alkalmazás futtatható Android API level 34+ (Android 14+) operációs rendszerű eszközökön, melyek rendelkeznek [ARCore 1.48](https://github.com/google-ar/arcore-android-sdk/releases/download/1.48.0/Google_Play_Services_for_AR_1.48.0.apk)-as verziójával.
Fontos, hogy csak 1.48-as verzión teszelt, és 1.52-es verzión kifejezetten hibás működést produkál. Részeletekért a [kibocsájtott APK leírását](https://github.com/L4Z4IX/ARnavUserApp/releases/tag/1.0) olvassa át a hiba kikerüléséért.
